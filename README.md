# 📈 SCT_DA_4 - Business Insights Report (EDA)

## 📌 Project Overview

This project was completed as part of the SkillCraft Technology Data Analytics Internship. The objective was to perform Exploratory Data Analysis (EDA) on a marketing campaign dataset and generate actionable business insights.

## 🎯 Objectives

* Clean and prepare the dataset
* Analyze customer demographics and spending behavior
* Evaluate campaign performance
* Identify high-performing purchase channels
* Generate business recommendations

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📈 Analysis Performed

* Data Cleaning & Preprocessing
* Customer Age Distribution Analysis
* Income vs Spending Analysis
* Purchase Channel Analysis
* Campaign Response Analysis
* Product Category Spending Analysis

## 📂 Files Included

* Marketing Campaign Dataset
* EDA Notebook
* Project Report
* Visualizations
* README.md
